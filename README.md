Learning Note of SMACSS (Scalable and Modular Architecture for CSS)
====================================================================

## SMACSS is
  - Categorization
  - Naming Convention
  - Decoupling CSS from HTML
  - State-based Design (How to build a webpage)

#### Categorization
  - Base (Recommend no css resets or normalize)
  - Layout
  - Module
  - State
  - Themes
